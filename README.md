# 🎓 Академический ассистент РАНХиГС (Telegram Bot)

Интеллектуальный Telegram-бот для автоматического проведения научных исследований, генерации структурированных текстовых отчетов (в формате MS Word `.docx`) и создания презентаций (в формате MS PowerPoint `.pptx`) с использованием многоагентных систем.

Проект построен на базе фреймворка **CrewAI** и использует умный балансировщик моделей для оптимизации затрат токенов и обхода лимитов API.

---

## 🚀 Два режима работы: Локальный и Облачный

Проект спроектирован так, чтобы работать в двух разных средах. Код проекта универсален, а переключение режимов происходит автоматически через переменные окружения.

### 💻 1. Локальный запуск (Гибридный режим / Hybrid) — Рекомендуется для работы
Используется на вашем персональном компьютере.
*   **Как работает**: Исследовательские агенты работают через встроенный прокси-сервер **FreeDeepseekAPI**, который бесплатно перенаправляет запросы в веб-интерфейс `chat.deepseek.com` (модели DeepSeek V3 и DeepSeek Reasoner/R1). Маршрутизатор (Selector) и дизайнер используют быстрый API Google Gemini.
*   **Плюсы**: Абсолютно бесплатно, нет ограничений на количество токенов и суточные лимиты, доступна мощная логика рассуждения R1.
*   **Почему работает локально**: Ваш домашний/рабочий IP-адрес является резидентным, поэтому Cloudflare на стороне DeepSeek не блокирует его.

### ☁️ 2. Облачный запуск (Режим Gemini) — Для Hugging Face Spaces
Используется при развертывании в контейнере Docker на сервере.
*   **Как работает**: Все агенты используют официальные API-ключи Google Gemini.
*   **Плюсы**: Высокая скорость работы, бот доступен 24/7 в облаке, не требует включенного компьютера.
*   **Почему в облаке только Gemini**: Серверные IP-адреса хостингов (таких как Hugging Face, AWS или Cloudflare) жестко блокируются защитой DeepSeek от парсинга. Запуск веб-прокси DeepSeek в облаке приведет к постоянным таймаутам.

## 🛠️ Быстрый старт: Локальный запуск (Windows)

### 1. Подготовка окружения
1. Убедитесь, что у вас установлены **Python 3.10–3.11** и **Git**.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Corebeat1431/ranepa-study-assistant.git
   cd ranepa-study-assistant
   ```
3. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```

### 2. Настройка конфигурационных файлов
Проект использует два конфигурационных файла, которые исключены из Git (`.gitignore`) для вашей безопасности:

1. **Создайте файл `.env`** в корневой папке проекта по шаблону:
   ```env
   # Выбор провайдера (hybrid — локально, gemini — в облаке)
   LLM_PROVIDER=hybrid

   # API Ключ для Google Gemini (получите бесплатно в Google AI Studio)
   GEMINI_API_KEY=ваш_ключ_gemini

   # Токен вашего Telegram-бота (получите у @BotFather)
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   ```

2. **Настройте сессию DeepSeek**:
   * Перейдите в папку `FreeDeepseekAPI`.
   * Скопируйте файл `auth.example.json` и назовите его `deepseek-auth.json`.
   * Авторизуйтесь на сайте [chat.deepseek.com](https://chat.deepseek.com) в вашем браузере.
   * Экспортируйте ваши авторизационные куки и токен авторизации (для этого в папке `FreeDeepseekAPI/chrome-extension` есть готовое расширение для Chrome, либо скопируйте значения заголовков `Cookie` и `Authorization Bearer` из вкладки Network в инструментах разработчика).
   * Заполните полученные данные в `deepseek-auth.json`:
     ```json
     {
       "token": "значение_bearer_токена_без_слова_Bearer",
       "cookie": "полная_строка_cookie_из_запроса",
       "wasmUrl": "https://chat.deepseek.com/assets/..." 
     }
     ```

### 3. Запуск
Просто дважды кликните по файлу **`Запустить_Ассистента.bat`** в корне проекта.
*   Скрипт активирует виртуальное окружение и запустит диспетчер (`launcher.py`).
*   Диспетчер переведет бота в спящий режим ожидания (Standby). При отправке команды `/start` в Telegram он автоматически поднимет прокси-сервер DeepSeek, запустит бота, а по завершении работы снова остановит процессы для экономии памяти вашего компьютера.

## ☁️ Развертывание в облаке (Hugging Face Spaces)

1. Создайте новое пространство (**Space**) на Hugging Face.
2. Выберите тип **Docker** (шаблон Blank или Dockerfile).
3. Настройте удаленный репозиторий и отправьте туда код:
   ```bash
   git remote add hf https://huggingface.co/spaces/ваш_логин/имя_пространства
   git push hf master:main
   ```
4. **Важно:** Перейдите в настройки Space (**Settings -> Variables and secrets**):
   * В разделе **Variables** создайте переменную:
     * `LLM_PROVIDER` со значением `gemini`.
   * В разделе **Secrets** добавьте секретные ключи:
     * `TELEGRAM_BOT_TOKEN` — токен вашего Telegram-бота.
     * `GEMINI_API_KEY` — ваш рабочий API-ключ Google Gemini.

## 📂 Структура проекта

*   `bot.py` — Главный файл Telegram-бота, обрабатывающий команды пользователей и координирующий запуски агентов.
*   `launcher.py` — Легковесный диспетчер процессов для локального запуска, переводящий бота в Standby-режим.
*   `src/llm_balancer.py` — Модуль балансировки и маршрутизации запросов между ИИ-моделями на основе ролей агентов.
*   `src/diagram_generator.py` — Логика построения векторных графиков для отчетов с поддержкой корректных шрифтов в Linux.
*   `src/pptx_generator.py` — Создание презентаций на базе Python-pptx с пропорциональным масштабированием слайдов.
*   `FreeDeepseekAPI/` — Локальный прокси-сервер на Node.js для работы с веб-версией DeepSeek.
*   `node-portable/` — Портативная версия Node.js для автономной работы прокси-сервера без необходимости установки Node.js в систему.
*   `requirements.txt` / `Dockerfile` — Зависимости и конфигурации для развертывания проекта в Docker.
